vemoji是一个基于vue框架的emoji键盘组件 方便用户输入emoji表情


## npm

https://github.com/wlor0623/v-emoji

## 用法

### cdn引用
```html
  <script src="https://cdn.jsdelivr.net/npm//v-emoji/dist/v-emoji.min.js"></script>

```
### 单文件引用
```js
import VEmoji from "v-emoji";

export default {
 components: {
    "v-emoji": VEmoji,
  },
}

```
